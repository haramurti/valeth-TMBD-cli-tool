# Panduan Kontribusi (Contributing Guide)

Halo! Terima kasih sudah mau mampir dan berkontribusi di project **TMDB CLI Tool**.

Gue sangat menghargai setiap bantuan, entah itu benerin *typo*, lapor *bug*, atau nambahin fitur baru biar alat ini makin canggih.

1.  **Fork Repo Ini**
    Klik tombol "Fork" di pojok kanan atas biar lo punya salinan project ini di akun lo sendiri.

2.  **Clone ke Laptop Lo**
    ```bash
    git clone [https://github.com/username-lo-disini/tmdb-cli.git](https://github.com/username-lo-disini/tmdb-cli.git)
    ```

3.  **Bikin Branch Baru**
    Jangan koding di branch `main` ya. Bikin cabang baru sesuai fitur yang mau lo buat.
    ```bash
    git checkout -b fitur-keren-baru
    ```

4.  **Mulai Koding**
    Silakan berkreasi! Tapi tolong perhatikan:
    - Gunakan `go fmt` sebelum commit biar kodingannya rapi.
    - Pastikan tidak ada error saat dijalankan.
    - Jangan sertakan API Key lo di dalam commit!

5.  **Push & Pull Request**
    Kalau sudah yakin, push ke repo lo dan bikin **Pull Request (PR)** ke repo asli gue.
    Jelasin di deskripsi PR: Apa yang lo ubah dan kenapa itu penting.
